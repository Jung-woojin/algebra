## 2.1 `1.1 행렬 A의 열을 이용한 곱셈 Ax`

핵심 질문:
- 왜 `Ax`를 "행렬-벡터 곱"이 아니라 "열의 조합"으로 봐야 하나?

30초 직관:
- `A=[a1 ... an]`, `x=[x1 ... xn]^T`이면  
  `Ax = x1 a1 + ... + xn an`  
  즉, `x`는 각 열벡터의 가중치다.

꼭 기억할 식:
- `Ax = sum_i x_i a_i`
- `Col(A) = {Ax | x in R^n}`

세부 포인트 매칭:
- `1.1.1` 독립인 열과 랭크: 독립한 열 개수 = rank

2분 체크:
- `A`의 열 3개가 모두 독립이면 rank는?
- `b`가 `Col(A)`에 없으면 `Ax=b`는 어떤 상태?

## 2.2 `1.2 행렬 곱셈 AB`

핵심 질문:
- `AB`를 빠르게 이해하는 가장 좋은 관점은?

30초 직관:
- `AB`의 j번째 열 = `A` * (`B`의 j번째 열)
- 즉, `B`가 만든 방향을 `A`가 다시 변환한다.

꼭 기억할 식:
- `(AB)_j = A b_j`
- `(AB)_{ij} = sum_k a_{ik} b_{kj}`

세부 포인트 매칭:
- `1.2.1` `AB`는 랭크 1 행렬의 합
- `1.2.2` 열 관점(열 조합)과 행 관점(내적)의 연결

2분 체크:
- `rank(AB) <= min(rank(A), rank(B))`가 왜 직관적으로 맞는가?

## 2.3 `1.3 네 가지 기본 부분공간`

핵심 질문:
- 왜 부분공간 4개(열/행/영공간/좌영공간)를 한 번에 봐야 하나?

30초 직관:
- 입력공간과 출력공간에서 "살아남는 방향"과 "사라지는 방향"을 동시에 추적하는 도구다.

꼭 기억할 식:
- `dim Col(A) = dim Row(A) = rank(A)`
- `dim N(A) = n - rank(A)`
- `dim N(A^T) = m - rank(A)` (A가 m x n)

세부 포인트 매칭:
- `1.3.1` `AB`, `A+B`의 랭크 비교 감각

2분 체크:
- `A`가 `5x3`, rank=2이면 `N(A)` 차원은?

## 2.4 `1.4 소거법과 A = LU`

핵심 질문:
- 왜 LU가 `Ax=b` 실전 계산의 기본인가?

30초 직관:
- 소거법은 `A`를 상삼각 `U`로 바꾸는 과정.
- 그때의 소거계수를 모으면 하삼각 `L`.
- 그래서 `A=LU`가 된다.

꼭 기억할 식:
- `A = LU`
- `Ax=b -> LUx=b`
- `Ly=b`(전진대입), `Ux=y`(후진대입)

세부 포인트 매칭:
- `1.4.1` 소거법으로 `Ax=b`
- `1.4.2` LU 분해
- `1.4.3` 해법 파이프라인
- `1.4.4` 행 교환이 있으면 `PA=LU`

2분 체크:
- 피벗이 0이면 왜 행 교환이 필요한가?

## 2.5 `1.5 직교행렬과 부분공간`

핵심 질문:
- 직교가 왜 계산과 안정성에서 중요할까?

30초 직관:
- 직교행렬 `Q`는 길이/각도를 보존하는 회전(또는 반사).
- 수치적으로 안정적이고 역행렬이 쉽다.

꼭 기억할 식:
- `Q^T Q = I`, `Q^{-1} = Q^T`
- `||Qx|| = ||x||`

세부 포인트 매칭:
- `1.5.1` 직교기저 = 좌표축 재정의
- `1.5.2` 하우스홀더 반사: QR 분해의 핵심 연산

2분 체크:
- 직교행렬의 열벡터들은 어떤 관계를 가지는가?

## 2.6 `1.6 고윳값과 고유벡터`

핵심 질문:
- "방향은 유지되고 크기만 바뀌는 벡터"를 왜 찾나?

30초 직관:
- 반복변환에서 지배적인 방향(모드)을 잡아낸다.
- 동적 시스템/그래프/최적화 안정성 판단에 직결.

꼭 기억할 식:
- `Av = lambda v`
- `det(A - lambda I)=0`

세부 포인트 매칭:
- `1.6.1` 직접 계산
- `1.6.2` 닮음행렬: `B=P^{-1}AP`는 같은 고윳값
- `1.6.3` 대각화: `A=PDP^{-1}`
- `1.6.4` 대각화 불가: 고유벡터 부족

2분 체크:
- 대각화 가능 조건을 한 줄로 말해보기.

## 2.7 `1.7 대칭 양의 정부호(SPD)`

핵심 질문:
- 왜 ML/최적화에서 SPD가 반복 등장할까?

30초 직관:
- `x^T S x > 0`이면 모든 방향 에너지가 양수.
- 손실 곡면이 "볼록한 그릇"에 가까워 계산이 안정적.

꼭 기억할 식:
- `S=S^T`, `x^T S x > 0 (x!=0)`
- 고윳값 모두 양수

세부 포인트 매칭:
- `1.7.1` 대칭행렬은 직교 고유벡터/실수 고윳값
- `1.7.2~1.7.5` SPD 판정(고윳값/주인수/추축)
- `1.7.6` `S=A^T A`는 항상 PSD, full column rank면 SPD
- `1.7.7~1.7.8` 최소화와 ML
- `1.7.9` 이차곡선(타원) 해석

2분 체크:
- `A^T A`가 왜 항상 음이 아닌지 설명해보기.

## 2.8 `1.8 특잇값 분해(SVD)`

핵심 질문:
- 왜 "아무 행렬"도 SVD로 분해 가능한가?

30초 직관:
- `A = U Sigma V^T`
- `V`로 입력축 회전 -> `Sigma`로 축별 스케일 -> `U`로 출력축 회전

꼭 기억할 식:
- `A = U Sigma V^T`
- `sigma_i = sqrt(lambda_i(A^T A))`

세부 포인트 매칭:
- `1.8.1` 축소형 SVD
- `1.8.2` 에카트-영 핵심 사실
- `1.8.3~1.8.4` 대수/기하 증명 아이디어
- `1.8.5~1.8.7` 첫 특이벡터, `A^T`의 특이벡터, 대칭 블록행렬 관점
- `1.8.8` `AB`와 `BA`의 0이 아닌 고윳값 공유
- `1.8.9` 부분행렬과 특잇값
- `1.8.10~1.8.11` 미분/적분, 유한차분 연산자와 SVD
- `1.8.12` 극분해 `A=QS`

2분 체크:
- PCA와 SVD의 연결을 한 줄로 말해보기.

## 2.9 `1.9 주성분과 최적의 낮은 랭크`

핵심 질문:
- 왜 `A_k`(rank-k 근사)가 데이터 압축의 표준 해답인가?

30초 직관:
- 정보가 많이 담긴 축(큰 특잇값)만 남기면 손실을 최소화한다.

꼭 기억할 식:
- `A_k = sum_{i=1..k} sigma_i u_i v_i^T`
- 에카트-영: 모든 rank-k 행렬 중 `A_k`가 최적

세부 포인트 매칭:
- `1.9.1~1.9.4` 에카트-영과 거리 최소화
- `1.9.2~1.9.3` 프로베니우스 노름 버전
- `1.9.5~1.9.8` PCA의 통계/기하/선형대수 해석
- `1.9.9` 0-1 행렬 특징

2분 체크:
- "분산 최대 축"과 "재구성 오차 최소"가 왜 같은 문제인지 설명.

## 2.10 `1.10 레일리 몫과 일반화된 고윳값`

핵심 질문:
- 왜 `Av=lambda Mv` 꼴이 실제 데이터 문제에서 자주 나오나?

30초 직관:
- 기준 내적이 유클리드가 아니라 `M`-가중 내적일 때의 고유문제다.
- 분류(LDA), 진동/물리계, 제약 최적화에서 표준 형태.

꼭 기억할 식:
- 일반화 고유값: `Av = lambda Mv`
- 레일리 몫: `R(x)= (x^T A x)/(x^T M x)`

세부 포인트 매칭:
- `1.10.1` 일반화 고유값 문제
- `1.10.2` M-직교성
- `1.10.3` `M`이 준정부호/비가역일 때 처리
- `1.10.4` 일반화 SVD 간소 형태
- `1.10.5` Fisher LDA 연결

2분 체크:
- LDA에서 분모/분자가 각각 어떤 산포를 의미하는지 쓰기.

## 2.11 `1.11 벡터/함수/행렬 노름`

핵심 질문:
- 노름을 바꾸면 "최적해"가 왜 달라지나?

30초 직관:
- 노름은 거리의 규칙이다.
- 규칙이 달라지면 벌점 구조가 달라지고 해석이 달라진다.

꼭 기억할 식:
- 벡터: `||x||_2`, `||x||_1`, `||x||_inf`
- 행렬: `||A||_F = sqrt(sum a_ij^2)`
- 스펙트럴 노름: `||A||_2 = sigma_max`

세부 포인트 매칭:
- `1.11.1` 직선 위 최적화 직관
- `1.11.2~1.11.4` 내적, 각, 함수공간 내적
- `1.11.5` 매끈함
- `1.11.6~1.11.7` 프로베니우스/유도노름
- `1.11.8` 핵 노름
- `1.11.9` 스펙트럼 반지름

2분 체크:
- L1 정규화가 희소해를 유도하는 이유를 기하로 설명.

## 2.12 `1.12 행렬/텐서 분해, 비음수, 희소`

핵심 질문:
- SVD 외에 NMF/텐서분해를 왜 써야 하나?

30초 직관:
- SVD는 부호가 섞여 해석성이 떨어질 수 있다.
- NMF는 "부분의 합"으로 해석이 쉽다.
- 텐서는 다차원 구조를 보존한다.

꼭 기억할 식:
- NMF: `X approx U V`, `U>=0, V>=0`
- CP: `T approx sum_r a_r ⊗ b_r ⊗ c_r`
- Tucker: `T approx G x1 A x2 B x3 C`

세부 포인트 매칭:
- `1.12.1~1.12.5` NMF와 최적 조건/계산
- `1.12.6` 희소 PCA
- `1.12.7~1.12.8` 텐서 기본/컬러이미지 텐서
- `1.12.9` `w=Av` 도함수
- `1.12.10` 결합확률 텐서
- `1.12.11` 텐서 랭크/노름
- `1.12.12~1.12.15` CP와 계산, 행렬화, 카트리-라오 곱
- `1.12.16` 터커 분해
- `1.12.17` 대형 텐서와 임의화

2분 체크:
- NMF와 SVD 중 얼굴 특징 해석에 더 직관적인 것은 무엇이고 왜 그런가?

## 3. 자주 막히는 포인트 12개

1. `Ax`를 행끼리 계산만으로 외우면 의미를 잃는다. (열 조합으로 보기)
2. `AB`에서 차원 확인을 먼저 안 하면 실수가 폭발한다.
3. 랭크와 독립열, nullity 관계를 함께 기억하지 않으면 문제를 못 푼다.
4. 소거법의 목표는 숫자계산이 아니라 구조(`LU`)를 얻는 것이다.
5. 직교행렬은 "길이 보존"이라는 기하 의미를 꼭 붙여서 기억하라.
6. 고유값은 "지배 방향"이고, SVD는 "아무 행렬의 표준형"이다.
7. `A^T A`는 항상 PSD다. SPD 조건은 rank와 연결된다.
8. PCA와 에카트-영은 사실상 같은 이야기를 다른 언어로 한 것이다.
9. 일반화 고유값은 내적공간이 바뀐 문제다.
10. 노름을 바꾸면 최적화 문제의 성질이 바뀐다.
11. NMF는 해석성, SVD는 최적성에 강점이 있다.
12. 텐서는 "차원을 눌러서 행렬로 푸는 법(행렬화)"이 핵심이다.

## 5. 마지막 한 페이지 요약

- 선형대수 1장은 결국 "변환(행렬) + 방향(벡터) + 에너지(이차형식) + 분해(SVD/PCA/NMF/텐서)" 이야기다.
- 실전에서는 다음 4가지를 우선 기억하면 된다:
  1. `A=LU` (해결)
  2. `A=U Sigma V^T` (분해)
  3. `x^T S x` (기하/최적화)
  4. `A_k` (저랭크 근사)

필요하면 다음 단계로, 이 보조자료의 각 절에 대해 `NumPy/PyTorch 코드 실습 버전`을 이어서 만들어줄 수 있다.
